# Prérequis pour installer mes dotfiles

## Paquets système essentiels

- hyprland
- waybar
- rofi
- swaync
- wlogout
- hyprlock
- hyprpaper

## Applications GUI

- kitty
- alacritty
- thunar
- firefox
- vesktop (Discord)
- spotify
- fastfetch
- nvim
- btop

## Outils CLI essentiels

- zsh
- eza
- bat
- fzf
- zoxide
- rclone
- docker
- flatpak
- grim
- wl-clipboard
- brightnessctl
- networkmanager
- polkit

## Audio

- pipewire (ou pulseaudio)
- wireplumber
- swayosd-server
- swayosd-client

## Plugins Zsh (installés automatiquement via zinit)

- powerlevel10k
- fzf-tab
- fast-syntax-highlighting
- zsh-completions
- zsh-autosuggestions

## Thèmes et polices

- Thème GTK : Orchis (inclus dans dotfiles)
- Polices : Nerd Fonts (pour powerlevel10k)

## Commandes pour installer (Arch Linux)

```bash
sudo pacman -S hyprland waybar rofi swaync wlogout hyprpaper \
  kitty alacritty thunar firefox fastfetch neovim btop \
  zsh eza bat fzf zoxide grim wl-clipboard brightnessctl \
  networkmanager polkit pipewire wireplumber swayosd \
  docker flatpak
```

```bash
# Installation de zinit et plugins (automatique via .zshrc)
# Docker
sudo systemctl enable docker


```
